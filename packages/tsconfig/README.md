# grafana-tsconfig

> Grafana's [TypeScript](https://typescriptlang.org) config.

## Installation

```shell
yarn add @grafana/tsconfig --dev
```

or:

```shell
npm install @grafana/tsconfig --save-dev
```

## Usage

```json
{
  "extends": "@grafana/tsconfig"
}
```

or:

```json
{
  "extends": "@grafana/tsconfig/base.json"
}
```
